# 🎓 Objetivos da Aula - Git & GitHub

> Nesta aula, você aprenderá os conceitos fundamentais de controle de versão com Git e GitHub através da prática.

---

## 📋 Roteiro Prático

### 1️⃣ Preparação Inicial

- ✅ **Criar uma conta no GitHub**
  - Acesse [github.com](https://github.com) e crie sua conta pessoal

---

### 2️⃣ Entendendo Repositórios

- 📦 **Criar um Repositório**
  - Conceito: Um repositório é um espaço seguro onde seu código é armazenado com histórico de versões
  
- 🔽 **Clonar o repositório**
  - Trazer o repositório remoto para sua máquina local
  ```bash
  git clone seu-link-aqui
  ```

---

### 3️⃣ Trabalhando com Branches

- 🌿 **Criando uma branch**
  - Branches são ramificações do código que permitem trabalhar paralelamente sem afetar a branch principal
  
- 🔀 **Alternando entre branches**
  ```bash
  git checkout nome-da-branch
  ```
  
- 🖥️ **Interface do GitHub no VS Code**
  - Explore a extensão Git integrada do VS Code para visualizar branches, commits e histórico

---

### 4️⃣ Criando Conteúdo

- 📄 **Criar um arquivo**
  - Crie um arquivo em linguagem Portugol para praticar

---

### 5️⃣ Adicionando no Repositório

- 📤 **Preparar e enviar suas mudanças**
  
  ```bash
  git add .                           # Adiciona todos os arquivos modificados
  git commit -m "Sua mensagem aqui"  # Registra as mudanças com uma mensagem
  git pull                            # Atualiza com mudanças remotas
  git push                            # Envia suas mudanças para GitHub
  ```

---

### 6️⃣ Contribuindo com Pull Request

- 🔄 **Criar um Pull Request (PR)**
  - Um PR é um pedido para mesclar suas mudanças na branch principal
  - Permite revisão de código antes da integração

---

### 7️⃣ Finalizando Mudanças

- 📝 **Alterar um arquivo**
  - Modifique um arquivo existente e repita o fluxo (add → commit → pull → push)

- ✨ **Completar/Mesclar Pull Request**
  - Revise, aprove e finalize a integração da sua contribuição

---

## 🎯 Resultado Final

Ao completar todos esses passos, você terá:
- ✔️ Uma conta ativa no GitHub
- ✔️ Experiência prática com Git
- ✔️ Conhecimento sobre branches e pull requests
- ✔️ Confiança para contribuir em projetos colaborativos
