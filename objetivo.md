# Objetivo da Aula
1. Criar uma conta no github
1. Criar um Repositório (O que é repositório)
   1. Clonar o repositório
      1. git clone seuLinkAqui
2. Criando branch
   1. git checkout nomeBranch
   2. Interface do github no VS Code
3. Criar um arquivo
   1. Utilizando Portugol
4. Adicionar esse arquivo no repositório
    1. git add .
    2. git commit -m "Sua mensagem aqui"
    3. git pull
    4. git push
5. Criando pull request
6. Alterando um arquivo
7. Completando pull request
